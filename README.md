# Fashion House 👗🛍️

Fashion House is a modern fashion e-commerce website built using HTML, CSS, Bootstrap, and JavaScript.  
The project showcases stylish UI components, responsive product sections, navigation menus, and shopping features for men’s and women’s fashion categories.

---

## 🚀 Features

- Responsive Bootstrap Layout
- Modern Navbar with Dropdown Menus
- Product Cards with Ratings
- Shopping Cart & Wishlist Icons
- Pagination System
- Stylish Footer Section
- Men & Women Fashion Categories
- Mobile Friendly Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- Bootstrap 5
- Bootstrap Icons
- JavaScript

